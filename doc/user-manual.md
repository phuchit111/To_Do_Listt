# TaskFlow User Manual / คู่มือการใช้งานระบบ TaskFlow

This manual provides detailed instructions for navigating and utilizing the TaskFlow Project Management System. Follow these guidelines to optimize your workflow and collaboration.

คู่มือนี้จัดทำขึ้นเพื่ออธิบายขั้นตอนการใช้งานระบบจัดการโครงการ TaskFlow โดยละเอียด โปรดปฏิบัติตามคำแนะนำเหล่านี้เพื่อเพิ่มประสิทธิภาพในการทำงานและการทำงานร่วมกันภายในทีม

---

## 1. Getting Started / การเริ่มต้นใช้งาน

### Account Registration / การลงทะเบียนบัญชีผู้ใช้
*   **English**: Access the application URL, navigate to the "Register" page, enter your details (Name, Email, Password), and click "Create Account".
*   **ภาษาไทย**: เข้าไปยัง URL ของแอปพลิเคชัน จากนั้นไปที่หน้า "Register" (ลงทะเบียน) กรอกข้อมูลพื้นฐาน (ชื่อ, อีเมล, รหัสผ่าน) และคลิกปุ่ม "Create Account" เพื่อเริ่มต้นใช้งาน

### User Authentication / การเข้าสู่ระบบ
*   **English**: Enter your registered email and password on the login page and select "Sign In". The system maintains sessions via secure tokens.
*   **ภาษาไทย**: กรอกอีเมลและรหัสผ่านที่ลงทะเบียนไว้ในหน้า Login จากนั้นเลือก "Sign In" ระบบจะรักษาการเชื่อมต่อของคุณผ่านระบบ Token ที่มีความปลอดภัยสูง

---

## 2. Workspace and Project Management / การจัดการเวิร์กสเปซและโครงการ

### Creating a Project / การสร้างโครงการ
*   **English**: Locate the "Projects" section in the sidebar, click "Add Project", specify a name, and select a color code for visual identification.
*   **ภาษาไทย**: ไปที่ส่วน "Projects" ในแถบเมนูด้านข้าง คลิก "Add Project" ระบุชื่อโครงการ และเลือกสีที่ต้องการเพื่อใช้ในการจำแนกโครงการด้วยสายตา

---

## 3. Task Management lifecycle / วงจรการจัดการงาน

### Creating a New Task / การสร้างงานใหม่
*   **English**: Click "Add Task" and fill in the title (mandatory), description, due date, priority, and category.
*   **ภาษาไทย**: คลิกปุ่ม "Add Task" และกรอกข้อมูลต่างๆ เช่น หัวข้องาน (จำเป็น), คำอธิบาย, วันที่กำหนดส่ง, ระดับความสำคัญ และหมวดหมู่ของงาน

### Managing Subtasks and Attachments / การจัดการงานย่อยและไฟล์แนบ
*   **English**: Use the task detail panel to add subtasks or upload relevant files for team collaboration.
*   **ภาษาไทย**: ใช้แผงรายละเอียดงานเพื่อเพิ่มงานย่อย (Subtasks) หรืออัปโหลดไฟล์ที่เกี่ยวข้อง เพื่อรองรับการทำงานร่วมกันภายในทีม

---

## 4. Viewing Modes / รูปแบบการแสดงผล

### List, Kanban, and Calendar Views / รูปแบบรายการ, คัมบัง และปฏิทิน
*   **English**: Switch between List view for filtering, Kanban for workflow management, and Calendar for deadline tracking.
*   **ภาษาไทย**: ท่านสามารถสลับรูปแบบการมองเห็นได้ระหว่าง รูปแบบรายการ (List) สำหรับการกรองข้อมูล, รูปแบบคัมบัง (Kanban) สำหรับการจัดการขั้นตอนการทำงาน และรูปแบบปฏิทิน (Calendar) สำหรับการติดตามกำหนดส่ง

---

## 5. Collaboration Features / ฟีเจอร์การทำงานร่วมกัน

### Comments and Tagging / การแสดงความคิดเห็นและการระบุตัวตน
*   **English**: Engage in discussions via comments and tag team members to assign responsibilities.
*   **ภาษาไทย**: สื่อสารภายในทีมผ่านระบบความเห็น (Comments) และใช้ระบบการระบุตัวตน (Tagging) เพื่อมอบหมายหน้าที่รับผิดชอบให้สมาชิกในทีม

---

## 6. Notifications and Alerts / ระบบแจ้งเตือน

### Automated Reminders / การแจ้งเตือนอัตโนมัติ
*   **English**: The system generates alerts 24 hours prior to a task's due date.
*   **ภาษาไทย**: ระบบจะดำเนินการแจ้งเตือนอัตโนมัติล่วงหน้า 24 ชั่วโมง ก่อนถึงกำหนดส่งของงานแต่ละชิ้น

---

## 7. System Customization / การปรับแต่งระบบ

### Theme Preferences / การตั้งค่าธีม
*   **English**: Toggle between Light and Dark modes using the icon at the bottom of the navigation sidebar.
*   **ภาษาไทย**: สามารถสลับระหว่างโหมดสว่าง (Light Mode) และโหมดมืด (Dark Mode) ได้ที่ไอคอนบริเวณด้านล่างของแถบเมนูข้าง ระบบจะจดจำค่าที่ท่านเลือกไว้โดยอัตโนมัติ
